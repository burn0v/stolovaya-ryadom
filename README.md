# CanteenNear

Сервис для поиска столовых, где можно посмотреть карту, почитать отзывы и оставить свою оценку.

## Структура репозитория

```text
.
├── backend/                # Основная часть на Django
│   ├── manage.py           # Запуск проекта
│   ├── canteennear/        # Логика приложения
│   │   ├── admin.py        # Настройка админки
│   │   ├── forms.py        # Формы (регистрация и т.д.)
│   │   ├── models.py       # База данных (столовые, отзывы)
│   │   ├── settings.py     # Конфигурация
│   │   ├── urls.py         # Ссылки и роутинг
│   │   ├── views.py        # Обработка запросов
│   │   └── templates/      # HTML шаблоны сайта
│   └── db.sqlite3          # Локальная база (только для разработки)
├── docs/                   # Документация
├── frontend/src/           # Фронтенд (статические файлы)
├── scripts/                # Полезные скрипты
├── ai/                     # Наработки по ИИ (в процессе)
├── .gitignore              # Игнорируемые файлы (скрываем мусор)
└── README.md               # Это описание
```

## Как запустить?
1. Установитиь зависимости. (Нужен Python 3.10 и выше)

```
pip install -r requirements.txt
```

2.Перейдите в папку с проектом и примените миграции:

```
cd backend
python manage.py migrate
```

3. Запустите локальный сервер.
```
python manage.py runserver
```
