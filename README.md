# Server Panel Project

## Структура репозитория (что куда и как надо класть)
**Папка "backend"** - в данной папке хранится все что связано с бэкендом, в том числе тесты кода, работа с api, источники (которые используются только в бэке) и т.п.

**Папка "frontend"** - в данной папке хранится все что связано с фронтедном, включая источники (которые используются только в фронте).

**Папка "docs"** - в данной папке находится вся документация, связанная с проектом, в том чилсле связанная с api, с бэк и фронт эндами, ai, и т.д.

**Папка "scripts" ** - используется для хранения всех скриптов используемых в проекте.

**Папка "ai"** - используется для хранения модели искуственного интелекта, промптов и всего что связано с ии.
